Case 1

B = 0 and CD = 1

If B = 0 while C = 1 and D = 1, then B'D = 1. The simplified expression already outputs 1.

Case 2

B = 1 and CD = 1

If B = 1 while C = 1 and D = 1, then BC = 1. The simplified expression already outputs 1.

Mapping the general theorem to this example gives X = B, Y = C, and Z = D. That makes YZ = CD the consensus term.
