Moore vs Mealy State Machines

Why Diagram Notation Mirrors the Physical Hardware

"Why does the Mealy machine label transitions as [input]/[output] instead of writing the output inside the state bubble?"

Because the diagram notation maps directly to the physical wires. In a Moore machine, the output wire connects only from the flip-flops (memory), so we write it inside the bubble. In a Mealy machine, the output wire bypasses memory and connects directly from the input through combinational logic — so we write it on the arrow.

The Bubble = Memory

A state bubble represents the D Flip-Flops in hardware. Memory only updates on a clock edge. Moore's output lives here because it's permanently wired to memory — it can only change when the clock ticks.

The Arrow = Combinational Logic

A transition arrow represents the combinational logic path. In a Mealy machine, the input wire bypasses memory and reaches the output gates through this path — so the output is labeled on the arrow.

The Bypass Wire

Mealy's secret: a direct wire from input to output logic that bypasses the flip-flops entirely. Because this wire skips memory, the output reacts instantly — between clock edges — which is why Mealy can detect one cycle earlier.
